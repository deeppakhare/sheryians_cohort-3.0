# Conditional Stetments

### 1. if Statement
`let age = 18;
if (age >= 18) {
    console.log("Eligible to vote");
}`

### 2. if-else Statement
`let num = 5;
if (num % 2 === 0) {
    console.log("Even");
} else {
    console.log("Odd");
}`

### 3. if-else-if Statement

`let marks = 75;
if (marks >= 90) {
    console.log("Grade A");
} 
else if (marks >= 60) {
    console.log("Grade B");
} 
else {
    console.log("Grade C");
}`
