# Math Functions 
we are use these functions for ferforming math operatoins

>1. Math.abs() :- It conversts negative values to positive values.
>2. Math.ceil() :- When there is any number after decimal point then it print the next integer of the given no. 
>3. Math.floor :- When there is any number after decimal point then it print the give integer (print integer befores decimal point).
>4. Math.round() :- in decimal number's point when there is number is less then 5 then it operates as Math.floor and when there is number is 5 or greater then 5 then it operates like Math.ceil.
>5. Math.trunc() :- It removes the decimal numbers after the point.
>6. Math.pow() :- It use to find power values of given. Ex., Math.pow(2,3); (means 2 to the power 3)
>7. Math.sqrt() :- It use to find square root of given integer.
>8. Math.cbrt() :- It use to find cube root of given integer.
>9. Math.max() :- It use to find maximun number from given numbers.
>10. Math.min() :- It use to find minimun number from given numbers.
>11. Math.random() :- It is use to print random number from 0 to 1.
>12. .toFixed() :- It is used to print how many decimal numbers we need to print which is after point. (Ex., `let a = 22.3456; console.log(a.toFixed(2))` the output is **22.34**).
