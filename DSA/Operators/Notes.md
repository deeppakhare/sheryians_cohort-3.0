# Operators
There are fore types of Operators in JS are as follows -

## 1. Arithematic Operators
    1. +
    2. -
    3. *
    4. /
    5. % (modulo operator)

## 1. Relational Operators
    1. <
    2. >
    3. <=
    4. >=
    5. !=

## 1. Logical Operators
    1. &&
    2. ||

## 1. Unary Operators
    1. ++
    2. --

> we use unary operator as "pre" and "post" type.

` ++a; ` (it is a pre increment)

`a++;` (it is a post increment)

`--a;` (it is a pre decrement)

`a--;` (it is a post decrement)
